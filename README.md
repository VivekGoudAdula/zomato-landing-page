# Zomato Landing Page

Welcome to the Zomato Landing Page!  
This is a simple landing page project built using **HTML** and **CSS**.  
No AI tools were used in the creation of this project — it is entirely hand-coded for learning purposes.

## Features

- Clean, modern design inspired by Zomato's homepage
- Responsive layout for desktop and mobile screens
- Simple navigation and attractive visuals
- Pure HTML and CSS, easy to understand and modify

## Learning Goals

- Practice HTML structure and semantic elements
- Explore CSS styling, layout, and responsiveness
- Understand how to build a real-world landing page from scratch

## Getting Started

1. **Clone this repository:**

   ```bash
   git clone https://github.com/VivekGoudAdula/zomato-landing-page.git
   ```

2. **Open the project:**
   - Open the folder in your preferred code editor.
   - Open `index.html` in your browser to view the landing page.

3. **Customize:**
   - Modify the HTML and CSS files to practice and improve your skills.
   - Experiment with different colors, fonts, and layouts.

## Project Structure

```
zomato-landing-page/
├── index.html
├── style.css
```

- `index.html` — main page markup
- `style.css` — custom styles for the page

## Inspiration

This project is inspired by Zomato's homepage and is built entirely from scratch for practice purposes.
